== Description ==
Plugin que instala una imagen en la esquina superior derecha de nuestras webs mostrando una etiqueta de apoyo a nuestro equipo el Athletic Club de Bilbao.


== Installation ==

1. Descargar el archivo comprimido en formato zip.
2. Descomprimirlo en la carpeta `wp-content/plugins` o instalarlo a traves del panel de administración.
3. Ir a la sección de plugins y activarlo.
